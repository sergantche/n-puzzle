## Target theorem (4×4)

```lean
theorem solvability_four_four (cfg : Config) :
    Reachable cfg goal ↔ parityClass cfg = parityClass goal
```

Matches the README for even `M = 4`:  
`I(L) + r_B ≡ 1 (mod 2)` ⟺ reachability of `goal`.

### 9b.3 — closed mathematics

With blank fixed at `bottomRight`, legal moves permute `tileList` (horizontal: identity on `L`; vertical: adjacent transposition in `L`). The closed proof realizes every even permutation of the 15 tiles from `goal`.

The group-theoretic core in `TileReach.lean` is:

- a bottom-right corner 3-cycle macro (`cornerPerm`);
- a Hamiltonian blank circuit inducing a 15-cycle (`hamPerm`);
- relabeling lemmas in `TileRelabel.lean`, so realizable permutations are closed under multiplication and inverse;
- a block/primitivity argument showing the subgroup generated by `cornerPerm` and `hamPerm` is primitive;
- Mathlib's Jordan theorem: a primitive permutation group containing a 3-cycle contains `alternatingGroup`.

Key closed lemmas:

- **`TileSign.lean`:** `sign_tileListPerm_eq_neg_one_pow`, `invStat_even_iff_perm_alternating` (builds green).
- **`permRealizable_of_mem_alternating`** (`TileReach.lean`): every `σ ∈ alternatingGroup (Fin 15)` is realized by some `cfg` with blank at BR and `Reachable goal cfg`.
- **`reachable_goal_to_cfg_bottomRight`** (`TileReach.lean`): uses `permOfCfg` + `configOfTileList`.
- **`tiles_to_goal_at_bottomRight`** (`TileConnectivity.lean`): via `reachable_symm` and `reachable_goal_to_cfg_bottomRight`; base case `invStat = 0`; symmetry via `TileInverse`.
